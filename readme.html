# Linux Task Manager [![License: MIT](https://img.shields.io/badge/License-MIT-green.svg?logo=mit&logoColor=white&style=for-the-badge)](LICENSE) [![Python](https://img.shields.io/badge/Python-3776AB?logo=python&logoColor=white&style=for-the-badge)](https://www.python.org/) [![GTK](https://img.shields.io/badge/GTK-476A34?logo=gtk&logoColor=white&style=for-the-badge)](https://www.gtk.org/) [![PyPI](https://img.shields.io/pypi/v/psutil?logo=python&logoColor=white&style=for-the-badge)](https://pypi.org/project/psutil/) [![GitHub issues](https://img.shields.io/github/issues/yourusername/linux-task-manager?style=for-the-badge&logo=github)](https://github.com/yourusername/linux-task-manager/issues) [![GitHub forks](https://img.shields.io/github/forks/yourusername/linux-task-manager?style=for-the-badge&logo=github)](https://github.com/yourusername/linux-task-manager/network) [![GitHub stars](https://img.shields.io/github/stars/yourusername/linux-task-manager?style=for-the-badge&logo=github)](https://github.com/yourusername/linux-task-manager/stargazers) A lightweight GTK 3 application for Linux that provides real-time monitoring and management of system processes, CPU/RAM usage, disk partitions, and network interfaces. It features an intuitive sidebar navigation, searchable process list with kill functionality, and dynamic system info visualizations. Ideal for users seeking a simple yet powerful graphical task manager built with Python and psutil. --- ## Table of Contents - [Features](#features) - [Requirements](#requirements) - [Installation](#installation) - [Usage](#usage) - [Screenshots](#screenshots) - [Contributing](#contributing) - [License](#license) - [Contact](#contact) --- ## Features - **Processes** View running processes with PID, name, CPU %, and memory %. Search/filter processes by name or PID. Right-click context menu to end tasks with confirmation. - **System** Visual CPU and RAM usage bars updated in real-time. - **Disk** List mounted disk partitions with usage statistics. - **Network** Display network interfaces with IP addresses and data sent/received. - Responsive UI with sidebar navigation and dynamic content area. - Periodic UI refresh every 3 seconds. - Custom dark-themed CSS styling. --- ## Requirements - Python 3.x - GTK 3 (PyGObject) - [psutil](https://pypi.org/project/psutil/) Python package --- ## Installation ### 1.
