Linux Task Manager

A lightweight GTK 3-based system monitor and process manager for Linux

üöÄ Features

  • Processes: View and search running processes by PID or name, with CPU and memory usage.
  • Right-click to end tasks with confirmation dialog.
  • System: Real-time CPU and RAM usage visualized with level bars.
  • Disk: Shows mounted partitions with usage statistics.
  • Network: Displays network interfaces, IP addresses, and data sent/received.
  • Dark-themed, responsive UI with sidebar navigation.
  • Automatic refresh every 3 seconds for up-to-date info.

  • Python 3.x
